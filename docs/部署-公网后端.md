# 部署后端到 Render（公网 HTTPS，手机无需同一 Wi-Fi）

## 一键部署

1. 打开 [Render Dashboard](https://dashboard.render.com/) 并登录
2. 点击 **New → Blueprint**
3. 连接 GitHub 仓库：`lq17746762169-create/qiheAPP-1`（或你的 fork）
4. Render 会读取根目录 `render.yaml` 自动创建 `qihe-server` 服务
5. 在环境变量中设置：
   - `DIFY_API_KEY` = 你的 Dify Service API Key
6. 部署完成后复制公网地址，例如 `https://qihe-server-xxxx.onrender.com`

## 写入手机 App

编辑 `app/config.js`：

```javascript
window.__QIHE_API_BASE__ = 'https://qihe-server-xxxx.onrender.com';
```

然后重新构建 APK：

```bash
cd mobile && bash scripts/build-release.sh
```

## 本地 Docker 运行

```bash
cd server
docker build -t qihe-server .
docker run -p 3000:3000 -e DIFY_API_KEY=app-xxx qihe-server
```
